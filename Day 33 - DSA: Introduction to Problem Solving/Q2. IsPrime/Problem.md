Problem Name: Q2. IsPrime

## Problem Description

Given a number A. Return 1 if A is prime and return 0 if not.

Note :
The value of A can cross the range of Integer.

## Problem Constraints

1 <= A <= 109

## Input Format

The first argument is a single integer A.

## Output Format

## Example Input

Input 1:
A = 5
Input 2:
A = 10

## Example Output

Output 1:
1
Output 2:
0

Example Explanation

## Example Explanation

Explanation 1:

5 is a prime number.

Explanation 2:

10 is not a prime number.
