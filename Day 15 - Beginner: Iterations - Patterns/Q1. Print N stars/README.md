Problem Name: Q1. Print N stars

## Problem Description

Given an integer N, print N stars in a single line.

For example if N = 5 then pattern will be like:

*****

## Problem Constraints

2 <= N <= 100

## Input Format

Single line input contains a single integer N.

## Output Format

Output N stars in a single line.

## Example Input

Input 1:

2

Input 2:

3

## Example Output

Output 1:

**

Output 2:

***
